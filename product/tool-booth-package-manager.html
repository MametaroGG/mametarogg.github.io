BOOTH Package Manager

Unity内だけでUnityPackageを完結管理。
エクスプローラーを開く手間を省き、プロジェクトを横断してアセットを瞬時に導入できる管理ツールです。

概要

「BOOTH Package Manager」は、数多くのUnityPackageを一元管理し、Unityエディタ内から直接プロジェクトへインポートできるようにするツールです。
管理データは外部のJSONファイルとして保存されるため、プロジェクトを切り替えても設定を引き継ぐことが可能。言語も日本語、英語、韓国語、中国語に対応しており、グローバルな環境でも使用いただけます。

主な機能

  • 横断管理: JSONファイルによる設定保存で、複数のプロジェクトから同じリストへアクセス。
  • 多言語対応: 日本語・英語・韓国語・中国語（簡体字）に完全対応。
  • 統計表示: ショップごとの導入統計をグラフィカルな棒グラフで確認可能。
  • 直感的なインポート: フォルダ構成を維持したまま、リストから選ぶだけで素早く導入。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Unity 2022.3.22f1 (VRChat推奨バージョン)
  • ✔ VRChat SDK 3.0 (導入済み環境で開発・動作確認済み)

注意点

※多言語対応の翻訳にはAIを使用しているため、一部不自然な箇所がある場合があります。
※外部JSONファイルにパスを保存する形式のため、ファイルを移動した場合は再指定が必要になります。
