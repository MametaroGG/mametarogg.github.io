PoseMenuBuilder

ポーズのメニューを無限に自動生成。
数多くのポーズやモーションを、Modular Avatarを利用して簡単にExpression Menuへ追加できるツールです。

BOOTHで購入する

概要

「PoseMenuBuilder」は、アバターのポーズメニュー作成を自動化するUnityEditor拡張です。
「フェイスメニュービルダー」のポーズ版として開発され、静止ポーズだけでなく動きのあるエモートの登録にも対応しています。手作業での複雑なメニュー構築からクリエイターを解放します。

主な機能

  • メニュー自動生成: 大量のポーズをExpression Menuへ自動で一括登録。
  • モーション対応: 静止ポーズだけでなく、アニメーションモーションの適応も可能。
  • Head/Handカスタマイズ: 頭部や手の動きを自由に制御・固定するためのカスタマイズ項目を搭載。
  • Modular Avatar連携: 非破壊的な改変をサポート。アバター本体を汚さずにメニューを追加できます。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Unity 2022.3.22f1 (VRChat推奨バージョン)
  • ✔ VRChat SDK 3.0 (導入済み環境で開発・動作確認済み)
  • ✔ Modular Avatar 1.13.4 以上

注意点

※アイコンは自動生成されません。アイコンを設定したい場合は「アイコンスタジオ」を使用するか、ご自身で画像を用意していただく必要があります。
