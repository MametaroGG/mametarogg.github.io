Boopa - BOOTH Price Tracker for VRChat

BOOTHのVRChat関連商品の価格推移をグラフで表示。
セール情報を見逃さず、賢く買い物を。アバターや衣装、ツールの価格履歴を商品ページ上に直接表示するブラウザ拡張機能です。

Chrome Web Store

概要

「あのアバター、前はいくらだったっけ？」を解決。
Boopa - BOOTH Price Tracker for VRChatは、BOOTHで販売されているVRChat向けアバターや衣装、ツールの価格推移を商品ページ上に直接グラフ表示するブラウザ拡張機能です。

主な機能

  • 価格推移グラフの表示: 商品ページを開くだけで、過去の価格履歴が美しいチャートで表示されます。
  • マルチバリエーション対応: アバターの「Quest版」や「衣装○○対応」など、複数の販売形式がある商品でもそれぞれの価格推移を個別に確認可能。
  • インタラクティブな操作: グラフ下の凡例にマウスを乗せると、特定のバリエーションだけを強調表示。クリックすると特定のグラフだけ非表示にすることができるので複雑な価格設定も一目で理解できます。
  • セール検知機能: セール期間中の価格変動も記録。過去の最安値をチェックして、次のセールの参考に。
  • 期間切り替え: 「直近5日間」から「過去5年間」まで、柔軟な期間指定でトレンドを把握できます。

安心・安全

  • ログイン不要: BOOTHのアカウント情報は一切使用しません。
  • 軽量動作: 外部の公開データ（GitHub Pages）から価格情報を取得するため、ブラウザの動作を重くしません。
  • データの利用目的: ユーザーが閲覧している商品の価格情報を表示するために、商品IDのみを参照します。
  • 個人情報の収集: ユーザーの個人情報、閲覧履歴、クッキーなどは一切収集・送信しません。

対象カテゴリー（自動判別）

VRChat関連のタグがついた3Dモデル、衣装、ツール類を自動的に判別してトラッカーを表示します。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Google Chrome

注意点

  • 価格データの記録および日付の計算は、日本時間（JST）を基準に行われています。
  • データの収集は日本時間で 毎日 0時～5時、6時～11時、12時～17時、18時～23時 の合計4回実行される設定になっています（対象ページを全て周回したら予定より早く終了します）。
  • GitHub Actionsで収集している都合上、開始時間に遅延が生じる場合があり、データの反映が遅れることがあります。
