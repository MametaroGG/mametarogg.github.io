VCC Template Builder

初期搭載したいツールやアセットをテンプレート化。
新しいプロジェクトを作成するたびに発生する「いつもの導入」を自動化する効率化ツールです。

BOOTHでダウンロード（無料） VCC版（GitHub）

概要

「VCC Template Builder」は、Unityプロジェクトのセットアップを劇的に効率化するツールです。
毎回手動でインポートしていたツール（lilToon, ModularAvatarなど）を事前にテンプレート化しておくことで、プロジェクト作成時からそれらが含まれた状態でスタートできます。

主な機能

  • ツールのテンプレート化: lilToonやModularAvatarなどの常用ツールをあらかじめプロジェクトに含めることが可能。
  • 設定のカスタマイズ: UnityPackageとしてインポートしたEditor拡張などもテンプレート化。
  • VCC/ALCOM対応: VCC版テンプレートはALCOMからもアクセス・利用可能です。
  • 複数プロジェクトへの適用: 一度作成したテンプレートは、新しいプロジェクト作成時に何度でも再利用できます。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Unity 2022.3.22f1 (VRChat推奨バージョン)
  • ✔ VRChat SDK 3.0 (導入済み環境で開発・動作確認済み)
  • ✔ VRChat Creator Companion (VCC)
