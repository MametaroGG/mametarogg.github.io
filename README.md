# 豆々庵ツール解説サイト

VRChat / Unity向けツールを販売する「豆々庵」の公式解説・ポートフォリオサイトです。
商品の詳細な仕様や使い方、制作者の活動実績を掲載しています。

## 公開URL
(https://mametarogg.github.io/)

## サイト概要
- **商品一覧**: BOOTHで販売中のツールを一覧表示
- **商品詳細**: 各ツールの詳しい使い方、仕様、導入手順
- **ポートフォリオ**: 制作者の過去の制作物やスキルセット

## 新商品追加手順
1. `product/tool-template.html` をコピーして、新しいHTMLファイルを作成（例: `tool-new-item.html`）。
2. ファイル内の「商品名」「概要」「機能」「BOOTHリンク」などを書き換える。
3. `products.html` の商品一覧カード部分をコピーし、新しい商品の情報に書き換えてリンクを貼る。
4. 必要であれば画像を `assets/images/` に追加する。
5. GitHubにプッシュすると自動的に更新されます。
