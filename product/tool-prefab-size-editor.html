Prefab Size Editor

プレハブの見た目のサイズを自由に変更。
さらに回転機能「Prefab Roll」も同梱。プロジェクトの視認性を劇的に改善します。

概要

「Prefab Size Editor」は、Unityプロジェクト上のプレハブの表示サイズを一括・個別に変更し、整理を容易にするエディタ拡張ツールです。
本商品は、制作者（みかん様）の代行出品となります。プレハブが多くなり、見た目が煩雑になってしまったプロジェクトの視認性を高めるのに最適です。

主な機能

  • サイズ自由変更: プレハブの表示サイズを単体・複数選択で自在に変更。
  • フォルダ単位の管理: フォルダ内の全プレハブに対して一括でサイズ変更を適用。
  • PNG保存対応: 変更した見た目のプレハブを画像として保存可能。
  • Prefab Roll 同梱: 「SHIFT」キーでの手軽なプレハブ回転機能を追加。
