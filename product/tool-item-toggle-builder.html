ItemToggleBuilder

アイテムON/OFFメニューを全自動生成。
メッシュを直接検出し、アバターの構造を壊さずに切り替えメニューを構築できる効率化ツールです。

BOOTHで購入する

概要

「ItemToggleBuilder」は、アバターの着せ替えや小物切り替えメニューの作成を自動化するUnityEditor拡張です。
アバタールートからメッシュを自動検出し、オン・オフしたいものを選ぶだけでExpression Menuを生成。オブジェクト単位ではなくメッシュ単位での制御により、バグの少ない安定した動作を実現します。

主な機能

  • 全アバター対応: 特定のアバターに依存せず、あらゆるアバターで使用可能。
  • メッシュベース制御: 構造を崩しにくいメッシュ単位のON/OFF切り替え。
  • 非破壊的な改変: Modular Avatarを利用し、アバター本体のデータを書き換えずにメニューを追加。
  • 直感的な選択UI: リストからアイテムを選ぶだけの簡単操作。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Unity 2022.3.22f1 (VRChat推奨バージョン)
  • ✔ VRChat SDK 3.0 (導入済み環境で開発・動作確認済み)
  • ✔ Modular Avatar 1.13.4 以上

注意点

※Modular AvatarのUI言語が日本語以外の場合、正常に機能しない可能性があります。
※アイテム数に制限はありませんが、VRChatのパラメーター上限（256メモリ）を超えると動作しませんのでご注意ください。
