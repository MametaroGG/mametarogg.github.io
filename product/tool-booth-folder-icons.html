BoothFolderIcons

Unityフォルダに自由なアイコンを設定。さらに独自のアセット検索機能も搭載。
プロジェクトを横断した管理と、直感的なアセットアクセスを実現する効率化ツールです。

BOOTHで購入する

概要

「BoothFolderIcons」は、Unityプロジェクトのフォルダに画像アイコンを設定し、視認性を飛躍的に高めるエディタ拡張ツールです。
ワールド制作などで膨大になったフォルダの中から目的のアセットを見つけるのは一苦労ですが、本ツールならアイコンで直感的に判別可能。さらにUnity標準よりも高性能な独自検索機能を備えており、制作の時短に大きく貢献します。

主な機能

  • フォルダアイコン設定: BOOTH画像をフォルダに表示。設定はプロジェクト間で保持。
  • 高性能検索システム: 大規模なプロジェクトでもストレスなく探せる独自検索。
  • プロジェクト横断設定: JSONファイルでの保存により、異なるプロジェクト間で共有。
  • 親フォルダ固定表示: 親フォルダのアイコンを任意で固定でき、視認性にも配慮。

ギャラリー

導入に必要なもの / 動作環境

  • ✔ Unity 2022.3.22f1 (VRChat推奨バージョン)
  • ✔ VRChat SDK 3.0 (導入済み環境で開発・動作確認済み)

注意点

※プロジェクト内のアセット量によっては、検索インデックスの作成に時間がかかる場合があります。
※アイコン設定はプロジェクトごとに柔軟に変更可能ですが、元画像のパスが変更されると表示されなくなる場合があります。
